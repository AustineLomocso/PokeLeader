# PokéLeader — Pokemon Gym Leader Generator

## Overview
PokéLeader is an AI-powered web app where a user uploads a photo, answers personality questions, and gets a shareable Pokemon Gym Leader card.

## Tech Stack
- Diffusion model: Stable Diffusion 1.5
- Face preservation: IP-Adapter
- Art style: Pokemon Trainer LoRA
- Text generation: Claude API
- UI: Gradio
